Parâmetros de Laudo - DB

Aqui estão os parâmetros específicos para o apoio DB.

Utilize com atenção, e não deixe de avaliar todos campos encontrados para melhorar a experiência com os laudos.

Pâmetros utilizados:

##LAUDOCOMPLETO## - TIPO: AREA -> exibe conforme ordem recebida no XML

##LAUDOCOMPLETO2## - TIPO: AREA -> deixa o MATERIAL e METODO na parte superior dos laudos

##LAUDOCOMPLETOEDITAVEL_DB## - TIPO: AREA -> segue laudo estruturado definido em modelo dentro de Configuração> Modelos

##LAUDOCULTURA## - TIPO: AREA -> dedicado para Cultura

##LAUDOELETRO## - TIPO: AREA -> dedicado para Eletroforese

##LAUDOHEMOGRAMA## - TIPO: AREA -> dedicado para Hemograma

##LAUDOSEMREF## - TIPO: AREA -> utilizado para exames que só possui uma coluna de resultados, comum em Biopsias e Citologias

##LAUDOURINA## - TIPO: AREA -> dedicado para exame de Urina Tipo I

##LIBERADORAPOIO## - TIPO: TEXTO -> exibe o liberador do exame no apoio

##LOCALEXAME## - TIPO: TEXTO -> exibe o CNES do apoio (ativar botão Liberador DB em Configuração>Parâmetros)

##MATERIAL## - TIPO: TEXTO -> exibe o Material

##METODO## - TIPO: TEXTO -> exibe o Metodo

##NOTA## - TIPO: AREA -> exibe a Nota

##OBS## - TIPO: TEXTO -> exibe as observações (utilizar mais se necessário, ex ##OBS2##, ##OBS3##, etc)

##PERFILHEPATICO## - TIPO: AREA -> dedicado para Perfil Hepatico

##REF## - TIPO: AREA -> exibe as Referências

##REGIAO## - TIPO: AREA -> exibe a Região de Coleta enviada ao apoio
