Integração - AGHuse

Aqui está o uso e configuração da integração AGHuse.

Visão Geral A integração AGHUse é responsável pela importação de pacientes e exportação de resultados. A ativação da integração é feita pelo WorklabGestor. Regras da Integração • O de/para de exames é feito pelo lado do AGHUse.
• Os códigos enviados devem estar no padrão do Worklab.
• Não é realizado envio de PDF.
• Os resultados são enviados campo à campo. Parâmetros Necessários Parâmetro Descrição usuarioAghuse Usuário da integração tokenAghuse Token da integração Estrutura Necessária • integracao_aghuse
• integracao_aghuse_de_para Alteração de Campo Campo Tipo codigo_aghuse VARCHAR(5) DEFAULT NULL De/Para de Resultados É necessário configurar de/para para exames com laudo montado, exames de apoio e exames com resultados compostos. Observações • Exames compostos devem ser configurados manualmente na tela de de/para.
• Caso o código do exame venha diferente do padrão Worklab, a integração não localizará o exame corretamente.
