Obs:

É necessário salvar a configuração, fechar e abrir o aplicativo novamente. Em seguida, com o INSPECIONAR aberto, acessar à aba:
APLICATION> Storange> Local Storange> abrir e ir em FILE.

Aqui são necessário criar alguns campos extras, sendo eles: (usar escritas exatas)
CAMPO VALOR
codlab Cliente deve solicitar ao Wareline e nos informar o código (ex.: 11).
filtro Informar: where laboratorio='' (utilizando o mesmo número definido no codlab).
unidadePadrao Código da unidade no Worklab (preencher apenas se o cliente desejar uma unidade fixa).
convenioPadrao Código do convênio no Worklab (preencher apenas se o cliente desejar um convênio fixo).
localPadrao Código do local no Worklab (preencher apenas se o cliente desejar um local fixo).


Banco de Dados

Adicionar na tabela PACIENTE a coluna 'prescricao' como VARCHAR 30 e como NULL;

Adicionar na tabela `wareline_pedidos` possui ja no id 2079, apenas extrair a tabela;

Adicionar na tabela `webservice_request` possui ja no id 2079, apenas extrair a tabela;


Usuário Obrigatório

É obrigatório que tenha um usuário cadastrado no sistema como INTEGRACAO, mesmo nome para Login e Nome de usuário


De/Para de Exames

O De/Para dos exames são os nossos códigos de exames internos no Worklab, o laboratório coloca eles no sistema do terceiro


Uso do Integrador

O cliente só precisa manter o integrador ligado e os botões dele ligados (verde)
