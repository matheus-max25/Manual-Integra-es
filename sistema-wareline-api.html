Configuração Gestor

Após ativar a integração no Gestor, acessar a tela integracaoWareline.php e preencher as credenciais fornecidas pelo Wareline.


Parâmetro Descrição
username Usuário fornecido pelo Wareline.
password Senha fornecida pelo Wareline.
grant_type Tipo de autenticação utilizado pela API.
client_id Identificador do cliente.
client_secret Chave secreta fornecida pelo Wareline.

Cadastro de/Para
  • No menu Integrações → Configuração → De/Para Exames, cadastrar todos os exames que serão utilizados pela integração.

  • No cadastro de Unidades e Locais, preencher o campo Unidade Wareline.

  • No cadastro de Convênios, preencher o campo Código Wareline, utilizado para identificação dos convênios.

Observações
  • As credenciais (username, password, grant_type, client_id e client_secret) devem ser fornecidas pelo Wareline.

  • O cadastro do De/Para é obrigatório para que os exames sejam identificados corretamente durante a integração.

  • Os campos Unidade Wareline e Código Wareline devem ser preenchidos conforme os códigos utilizados pelo parceiro.

  • Após concluir toda a configuração, recomenda-se realizar um pedido de teste para validar a comunicação entre os sistemas.

Fluxo da Integração
Etapa Descrição
1 Ativar a integração pelo Gestor.
2 Executar os scripts de banco de dados.
3 Informar as credenciais da API Wareline.
4 Configurar o De/Para dos exames.
5 Configurar Unidade Wareline e Código Wareline dos convênios.
6 Realizar testes de envio e recebimento.
